# 🌑 Shadow‑Archives  
*A full‑stack web application for submitting and browsing articles and anomaly sightings.*

## 📌 Overview  
Shadow‑Archives is a full‑stack application built as the final project for the **Codecademy Full Stack Development Program**.  
It allows users to upload and view **articles** and **sighting reports**, stored in a PostgreSQL database and rendered through Handlebars templates.  

This project demonstrates full‑stack development skills including RESTful routing, MVC architecture, database modeling, and deployment on Render.

---

## 🛠️ Technologies Used  
### **Backend**
- Node.js  
- Express.js  
- RESTful API routes (GET, POST)  
- MVC architecture  
- Environment variables for security  

### **Frontend**
- Handlebars.js (templating engine)  
- Responsive UI  
- Custom CSS  

### **Database**
- PostgreSQL  
- Sequelize ORM *or* pg (depending on your implementation)  
- Two tables (Articles, Sightings)  
- Seed data + user‑generated entries  

### **Deployment**
- Render (Web Service + PostgreSQL Database)

### **Exploration Requirement**
- New frontend/CSS technology (cosmic UI, animated effects, or whichever exploration item you choose)

---

## 📂 Features  
- Submit new **articles**  
- Submit new **sighting reports**  
- View all entries stored in the database  
- Clean, responsive UI  
- Organized MVC structure  
- Secure environment variables  
- Fully deployed on Render  

---

## 🗄️ Database Structure  
### **Tables**
#### `articles`
- id  
- title  
- content  
- created_at  

#### `sightings`
- id  
- location  
- description  
- date  
- created_at  

Each table contains at least **5 seed records**, with support for user‑generated submissions.
